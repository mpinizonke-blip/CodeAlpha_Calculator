# CodeAlpha Task 2: Build a Calculator

A responsive calculator built with HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Internship.

## Features
- Basic arithmetic operations: +, -, *, /
- User-friendly interface with buttons and display screen
- Clear button to reset calculations
- Real-time result display with error handling
- Keyboard support: numbers, operators, Enter, Backspace, Escape
- Responsive design for mobile and desktop
- Hover effects and smooth transitions

## Technologies Used
- HTML5
- CSS3 - Flexbox & Grid
- JavaScript ES6

## How to Run
1. Clone this repository
2. Open `task2-calculator/index.html` in your browser
